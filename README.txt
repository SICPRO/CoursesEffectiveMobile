Courses Effective Mobile

Стек:
Язык - Kotlin
Архитектура - Clean Architecture + MVVM
Структура - Multi-module (feature_login, feature_main)
DI - Koin
Сеть - Retrofit 2 + OkHttp
БД - Room (KSP)
Навигация - Navigation Component
Async - Coroutines + LiveData
UI - XML / ViewBinding

Архитектура
Проект разбит на feature-модули, каждый из которых следует трёхслойной Clean Architecture:
app/ - Точка входа
feature_login/ - Экран авторизации
feature_main/ - Основной функционал приложения

Экран входа
- Валидация email по маске (regex) с блокировкой кириллицы
- Валидация пароля
- Вход через ВКонтакте и Одноклассники (ссылки для открытия в браузере)
- Навигация на главный экран после успешного входа

Главный экран
- Загрузка курсов через Retrofit с удалённого JSON
- Список курсов через RecyclerView + AdapterDelegate
- Сортировка по дате публикации (publishDate)
- Добавление/удаление курса из избранного с сохранением в Room
- Синхронизация состояния hasLike из JSON с локальной БД при запуске

Экран детали курса
- Полное описание курса
- Кнопка закладки с актуальным состоянием избранного

Экран избранного
- Список курсов добавленных в избранное из Room
- Удаление из избранного по клику на закладку
- Переход на детальный экран

Экран профиля
Макет-заглушка с меню
